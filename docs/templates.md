# Templates

Templates live in `templates/` and are copied by `mkproject` or `mkassign`.

## Basic

```text
my-app/
|-- src/
|-- docs/
|-- screenshots/
|-- README.md
`-- .gitignore
```

Use it for small tools, experiments, and general coursework projects.

## Python

```text
my-app/
|-- src/
|-- tests/
|-- docs/
|-- README.md
|-- .gitignore
`-- requirements.txt
```

Use it for scripts, APIs, and Python learning projects.

## Node

```text
my-app/
|-- src/
|-- public/
|-- docs/
|-- README.md
|-- .gitignore
`-- package.json
```

Use it for frontend experiments and small Node.js projects.

## Assignment

```text
assignment-name/
|-- questions/
|-- answers/
|-- screenshots/
|-- references/
|-- submissions/
`-- README.md
```

The README includes the assignment name, current date, and a task checklist.
